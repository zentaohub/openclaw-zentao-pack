## 第一阶段：禅道 + 企业微信 官方免登集成

### 1.1 企业微信后台配置

**操作人：** 企业微信管理员

1. 登录 **企业微信管理后台** https://work.weixin.qq.com

2. **应用管理** → **自建应用** → **创建应用**
   - 应用名称：`禅道`
   - 应用图标：上传禅道 logo
   - 可见范围：选择需要访问禅道的部门/人员

3. 记录以下信息（保存好）：
   ```
   企业 ID (CorpID): _______________
   应用 AgentID: _______________
   应用 Secret: _______________
   ```

4. 配置应用主页：
   - **应用主页 URL**: `https://zentao.yourcompany.com/`
   - （替换为你的禅道实际地址）

5. 配置可信域名：
   - **可信域名**: `zentao.yourcompany.com`
   - （需要 DNS 验证，按提示操作）

---

### 1.2 禅道后台配置

**操作人：** 禅道管理员

1. 登录 **禅道企业版后台**

2. **后台** → **集成** → **企业微信**

3. 填写配置：
   ```
   企业 ID (CorpID): [填入 1.3 中记录的企业 ID]
   应用 AgentID: [填入 1.3 中记录的 AgentID]
   应用 Secret: [填入 1.3 中记录的 Secret]
   ```

4. **启用"免登录"选项**

5. **启用"消息通知"**（可选，用于禅道通知推送到企微）

6. 保存配置

7. **测试免登**：
   - 在企业微信里找到"禅道"应用
   - 点击进入，应该直接跳转到禅道且已登录状态

---

### 1.3 用户账号匹配

**禅道用户名 = 企业微信用户名** → 自动匹配

如果不一致，管理员可以在：
- **禅道后台** → **组织** → **用户** → 编辑用户
- 填写"企业微信账号"字段进行手动绑定

## 🔧 故障排查

### 问题 1：企业微信进禅道还要登录

**原因：** 用户名不匹配

**解决：**
- 禅道后台 → 组织 → 用户 → 编辑
- 填写"企业微信账号"字段

---

### 问题 2：小禅说"未找到禅道凭证"

**原因：** TOOLS.md 配置缺失

**解决：**
- 检查 `/root/.openclaw/workspace/TOOLS.md` 是否有禅道 API 配置
- 确认服务账号密码正确

---

### 问题 3：小禅查询不到任务

**原因：** 用户映射缺失，或企业微信接口 IP 白名单未放行

**解决：**
- 在 TOOLS.md 的"禅道用户映射"表中添加映射
- 或者确保企微 UserID = 禅道用户名
- 如果要启用自动读取企业微信通讯录，需要在企业微信后台给 `1.14.73.166` 配置接口 IP 白名单
