# 进度查询说明

## 目标

为项目、执行和任务提供稳定的禅道进度查询与汇总方式。

## 查询模式

- `project`：项目整体进度
- `execution`：执行、Sprint 或迭代进度
- `task`：单个任务的完成情况

## 封装脚本输入

`scripts/get_progress.ts` 接收以下参数：

- `entity_type`: `project`, `execution`, or `task`
- `entity_id`
- `include_children`

## 输出摘要预期

返回一个简洁的 JSON 结果，至少包括：

- 目标实体 ID
- 实体类型
- 若存在则返回状态
- 若存在则返回完成百分比
- 若接口可提供，则返回总数、已完成数和剩余数等统计信息
